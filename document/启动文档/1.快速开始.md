# 快速开始

> [!IMPORTANT]
> 需要先启动后端服务：[Auth Matrix Backend](https://github.com/thirty30ww/auth-matrix-backend)

## 环境要求

- Node.js 16.0+
- npm 7.0+ 或 yarn 1.22+

## 安装依赖

```bash
cd frontend
npm install
```

配置镜像（可选）：
```bash
npm config set registry https://registry.npmmirror.com
```

## 启动开发

```bash
npm run dev
```

访问：http://localhost:5173

## 生产构建

```bash
npm run build
npm run preview
```

## 配置说明

修改后端地址（`vite.config.ts`）：

```typescript
server: {
  proxy: {
    '/api': {
      target: 'http://localhost:8080',
      changeOrigin: true
    }
  }
}
```
